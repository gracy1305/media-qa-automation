# Media QA Automation Demo

Built a Selenium-based automation script to validate media playback behavior in a web application.

## Features
- Automatically opens browser
- Loads local media page
- Detects video element
- Plays and pauses video
- Logs test results

## Tech Stack
- Python
- Selenium
- Chrome WebDriver

## Use Case
Simulates real-world QA testing for media-heavy applications like streaming platforms.
